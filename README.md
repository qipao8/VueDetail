### 2. 核心原理
#### 数据响应式核心
说人话就是：**页面用到一些数据，这些数据改变之后，页面上的数据也随之改变。** 用脚指头想想就知道，要实现响应式，我们需要主要解决下面这些问题：
1. 获取页面用到的数据，即数据劫持
2. 当数据变化能够知道，即数据监听
3. 数据更新到页面用到的位置，即依赖收集与分发
4. 数据变化影响到一些其他地方，需要进行一些其他操作，即响应操作
